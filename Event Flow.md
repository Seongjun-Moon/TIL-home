## 이벤트 플로우



#Login 

```
ID 입력
PW 입력
=> 로그인
```

#Home 화면

```
1. 안녕하세요. id님
팔로워 수	팔로잉 수
내프로필Button 로그아웃Button 표시

2. 게시글 업로드 기능
  2-1. 게시글 작성 칸에 글 작성(해시태그 추가 가능)
  2-2. 사진 업로드 시 사진업로드 버튼 클릭
  2-3. '짹짹' Button 클릭 시 업로드
  
3. 태그 검색 기능
 3-1. 보고싶은 태그 입력 후 검색 클릭
 3-2. 해당 태그 입력 된 게시글들 출력
 3-3. 해당 태그 입력 한 User 중 팔로우 하지 않은 User 팔로우 가능
```
